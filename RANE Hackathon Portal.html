Challenge Themes & Problem Statements

Industrial Automation

CNC LoadUnload Automation

Challenge: CNC machines require manual part loading/unloading, wasting time and resources.

Solution Needed: Design an automated system for CNC loading and unloading operations.

Industrial Automation

Automated Thread Checking System

Challenge: Manual thread inspection using gauges is slow and inconsistent.

Solution Needed: Develop an automated inspection system with AI-based quality verification.

Smart Manufacturing

Tool Life Prediction

Challenge: Predict tool life in broaching machines using IoT sensor data and ML algorithms.

Solution Needed: Build predictive models to optimize tool replacement cycles.

Advanced Materials

Anti-Carburization Solution

Challenge: Find cost-effective alternative to anti-carb paste for decarburising.

Solution Needed: Explore composite materials and coating solutions.

Advanced Materials

Composite Material Selection

Challenge: Identify cost-effective composite materials for safety-critical steering components.

Solution Needed: Material analysis and feasibility study for structural applications.

Advanced Materials

Material Stress-Strain Analysis

Challenge: Establish SN curves for processed/converted materials vs. standard materials.

Solution Needed: Develop comparative analysis for Indian standard materials.
